**#Proyecto del Sprint 8: Ejercicio**

El proyecto del Sprint 8, automatiza 9 pruebas funcionales para el sitio web de Urban Routes 
que es una app para solicitar transporte. 


_**#Las pruebas fueron las siguientes:**_
1- Configura una dirección de origen y destino. 
2- Selecciona la tarifa "Comfort". 
3- Ingresa un número de teléfono.
4- Ingresa el código SMS recibido.
5- Agrega una tarjeta de crédito.
6- Escribe un mensaje personalizado para el conductor.
7- Solicita manta y pañuelos. 
8- Pide dos helados.
9- Confirma el viaje y espera a que aparezca la información del conductor.


**_#Tecnologias y herramientas utilizadas_**
-Python 3.9+
-Page Object Model(POM)
-Selenium (WebDriver) 
-Pytest
-Google Chrome instalado
-ChromeDriver

_**#Como ejecutar la pruebas**_

-Actualizar la url en el archivo 'data.py' específicamente en la variable 'urban_routes_url', antes de ejecutar las pruebas.
-En PyCharm puedes hacer clic derecho sobre el archivo 'main.py' y posteriormente dar clic en la opción "Run Python tests in main".
-De igual manera se puede correr con la combinación de teclas Ctrl+Mayús+F10
